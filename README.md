# ⚽ Nova Geração FC - Landing Page

Uma landing page moderna e responsiva desenvolvida para o time de Fut7 **Nova Geração FC**, com foco em visibilidade, captação de patrocinadores e presença digital profissional.

---

## 🚀 Sobre o Projeto

Este projeto foi criado com o objetivo de representar o time de forma profissional na internet, destacando:

* 📸 Galeria de imagens do time
* 💼 Área dedicada para patrocinadores
* 📱 Integração com redes sociais
* 📞 Contato direto via WhatsApp

---

## 🛠️ Tecnologias Utilizadas

* ⚛️ React
* ⚡ Vite
* 🎨 Tailwind CSS
* 🎬 Framer Motion
